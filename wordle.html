Simple Python Wordle Game

Posted on November 09, 2024

Welcome back to The Data Chronicles!

Today, we're diving into the world of word games by exploring a simple yet fascinating Python script that brings the popular game Wordle to your command line.

We'll dissect the script step by step, unraveling how each part contributes to the game's functionality.

The Complete Script


import random

# List of 5-letter words
word_list = [
    'apple', 'brain', 'charm', 'delta', 'eagle', 'flame', 'grape', 'house', 'input', 'jolly',
    'knife', 'lemon', 'magic', 'night', 'ocean', 'piano', 'queen', 'robot', 'smart', 'train',
    'unity', 'vivid', 'whale', 'xenon', 'young', 'zebra'

# Choose a random word from the list
secret_word = random.choice(word_list)

MAX_ATTEMPTS = 6

print("Welcome to Wordle!")
print(f"You have {MAX_ATTEMPTS} attempts to guess the 5-letter word.")

for attempt in range(1, MAX_ATTEMPTS + 1):
    guess = input(f"Attempt {attempt}: ").lower()

    if len(guess) != 5:
        print("Please enter a 5-letter word.")
        continue

Step-by-Step Breakdown

1. Importing the Random Module

import random

The random module allows us to make random selections, which is essential for choosing a secret word from our list.

2. Defining the Word List

word_list = [ 'apple', 'brain', 'charm', 'delta', 'eagle', 'flame', 'grape', 'house', 'input', 'jolly', 'knife', 'lemon', 'magic', 'night', 'ocean', 'piano', 'queen', 'robot', 'smart', 'train', 'unity', 'vivid', 'whale', 'xenon', 'young', 'zebra' ]

We create a list of 5-letter words that the game will use both for selecting the secret word and validating player guesses.

3. Selecting the Secret Word

secret_word = random.choice(word_list)

random.choice() selects a random word from word_list.

Assigns a secret word that the player needs to guess.

4. Setting Maximum Attempts

MAX_ATTEMPTS = 6

Defines how many guesses the player is allowed, adding a challenge to the game.

5. Welcoming the Player

print("Welcome to Wordle!") print(f"You have {MAX_ATTEMPTS} attempts to guess the 5-letter word.")

Displays a welcome message and informs the player about the game's rules.

6. Starting the Game Loop

for attempt in range(1, MAX_ATTEMPTS + 1):

Creates a loop that runs once for each attempt.

range(1, MAX_ATTEMPTS + 1) generates numbers from 1 to 6.

7. Prompting for a Guess

guess = input(f"Attempt {attempt}: ").lower()

Prompts the player to enter a guess and converts it to lowercase for consistency.

8. Validating the Guess Length

if len(guess) != 5: print("Please enter a 5-letter word.") continue

Ensures the player enters a word of the correct length.

If not, it informs the player and restarts the loop without counting the attempt.

9. Checking if the Word is in the List

if guess not in word_list: print("Word not in list.") continue

Validates that the guess is a recognized word from our list.

If the word isn't in the list, it informs the player and restarts the loop.

12. Checking for a Correct Guess

13. Handling Exhausted Attempts

This else block executes if the loop completes without a break, meaning the player didn't guess the word.

Informs the player that the game is over and reveals the secret word.
